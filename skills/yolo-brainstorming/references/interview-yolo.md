# YOLO 访谈决策树

按维度逐题处置，已确认的跳过。

## Q1: 检测目标

- 单一类别 / 多类别？
- 目标尺寸分布（大/中/小目标比例）？
- 是否有遮挡场景？

## Q2: 数据集

- 已有标注？格式？
- 数据量？（< 100 / 100-1000 / 1000-10000 / > 10000）
- 是否需要数据增强？
- 训练/验证/测试划分比例？

## Q3: 性能要求

- 精度优先 / 速度优先 / 平衡？
- 目标 FPS？
- 目标 mAP？

## Q4: 部署平台

- GPU / CPU / NPU / MCU？
- 具体芯片型号？
- 功耗限制？
- 内存限制？

## Q5: 训练约束

- GPU 显存大小？
- 可训练时长？
- 是否需要恢复训练？

## Q6: 导出约束

- 必须支持的框架？
- 是否需要量化？
- 精度损失可接受范围？

## Q7: 集成方式

- Python API / C++ / REST API / 嵌入式 SDK？
- 是否需要实时视频流？
- 输入源类型（摄像头/图片/视频/RTSP）？
